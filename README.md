# 🍔 Food Delivery Website

A modern and responsive **Food Delivery Web Application** that allows users to browse food items, add them to the cart, and place orders easily.
The goal of this project is to provide a **smooth user experience with a clean and responsive UI**.

---

## 📌 Features

* Browse food items
* Add items to cart
* Remove items from cart
* Dynamic cart updates
* Responsive design for mobile and desktop
* Clean and modern UI
* Simple navigation between pages

---

## ⚙️ Installation

### 1️⃣ Clone the repository

```
git clone https://github.com/arshith-2909/FOOD-DELIVERABLE.git
```

### 2️⃣ Navigate to the project folder

```
cd frontend
```

### 3️⃣ Install dependencies

```
npm install
```

### 4️⃣ Run the development server

```
npm run dev
```

## 🚀 Usage

1. Open the website in your browser
2. Browse available food items
